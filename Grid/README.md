### Класс `Grid`

#### Публичные члены:

- `Grid()`: Конструктор по умолчанию. Создает пустую сетку.
- `explicit Grid(T const& t)`: Конструктор, инициализирующий сетку одним значением `t`.
- `Grid(size_type y_size, size_type x_size)`: Конструктор, создающий сетку заданного размера (`y_size` строк, `x_size` столбцов) и инициализирующий её значениями по умолчанию.
- `Grid(size_type y_size, size_type x_size, T const& t)`: Конструктор, создающий сетку заданного размера и инициализирующий все элементы значением `t`.
- `Grid(Grid<T> const& other)`: Конструктор копирования, создает копию переданной сетки.
- `Grid<T>& operator=(Grid<T> const& other)`: Оператор присваивания, выполняет копирование сетки.
- `Grid(Grid<T>&& other)`: Конструктор перемещения, осуществляет перемещение ресурсов из переданной сетки.
- `Grid<T>& operator=(Grid<T>&& other)`: Оператор перемещающего присваивания, перемещает ресурсы из другой сетки.
- `~Grid()`: Деструктор, освобождает память, занятую сеткой.
- `size_type get_y_size() const`: Возвращает количество строк в сетке.
- `size_type get_x_size() const`: Возвращает количество столбцов в сетке.
- `T& operator()(size_type y, size_type x)`: Позволяет получить доступ к элементу сетки по заданным координатам `(y, x)`.
- `GridRow operator[](size_type y)`: Позволяет индексировать сетку, возвращая объект класса `GridRow`, который позволяет использовать оператор `[]` для доступа к строке сетки по индексу `y`.
- `void print()`: Выводит содержимое сетки на экран.

#### Приватные члены:

- `T* data`: Указатель на массив данных сетки.
- `size_type y_size, x_size`: Размеры сетки (количество строк и столбцов).
- `class GridRow`: Вспомогательный класс, представляющий строку сетки.

### Класс `nGrid`

#### Публичные члены:

- `nGrid()`: Конструктор по умолчанию. Создает пустую n-мерную сетку.
- `explicit nGrid(size_type len, Args... coords)`: Конструктор, создающий n-мерную сетку заданного размера (`len`) с координатами `coords...` и инициализирующий её значениями по умолчанию.
- `explicit nGrid(size_type len, Args... coords, T const& t)`: Конструктор, создающий n-мерную сетку заданного размера и инициализирующий все элементы значением `t`.
- `nGrid(nGrid<T,n> const& other)`: Конструктор копирования, создает копию переданной n-мерной сетки.
- `nGrid<T,n>& operator=(nGrid<T,n> const& other)`: Оператор присваивания, выполняет копирование n-мерной сетки.
- `nGrid(nGrid<T,n>&& other)`: Конструктор перемещения, перемещает ресурсы из переданной n-мерной сетки.
- `nGrid<T,n>& operator=(nGrid<T,n>&& other)`: Оператор перемещающего присваивания, перемещает ресурсы из другой n-мерной сетки.
- `~nGrid()`: Деструктор, освобождает память, занятую n-мерной сеткой.
- `void clear()`: Очищает содержимое n-мерной сетки.
- `T& operator()(size_type y, Args... coords)`: Позволяет получить доступ к элементу сетки по заданным координатам.
- `nGrid<T,n-1>& operator[](size_type y)`: Позволяет индексировать n-мерную сетку и получить (n-1)-мерную сетку по индексу `y`.
- `void print()`: Выводит содержимое n-мерной сетки на экран.

#### Приватные члены:

- `nGrid<T,n-1>* data`: Указатель на массив n-мерных сеток.
- `size_type len`: Длина текущей n-мерной сетки.
